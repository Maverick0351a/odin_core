## 📦 Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "ODIN Protocol"
4. Click Install

Or install from VSIX:
```bash
code --install-extension odin-protocol-1.0.0.vsix
```
